This website is an implementation of a very popular dataset called the 'Food 101' dataset. However, the complete dataset is not used. Only a part of the dataset is used. Instead of 101 classes, only 10 classes are used. The following classes are present in the dataset and our model can identify only from these classes.

  1. Chicken Curry
  2. Chicken Wings
  3. Fried Rice
  4. Grilled Salmon
  5. Hamburger
  6. Ice Cream Pizza
  7. Pizza
  8. Ramen
  9. Streak
  10. Sushi

The model used is Resnet50_V2 which is a transfer learning model and achieved a validation accuracy of close to 84%. Any clarifications regarding the model structure/architecture can be found on my Github. Also, special credits to Mr Daniel Brouke who is the person behind creating this sub dataset of the 'Food 101' dataset.
